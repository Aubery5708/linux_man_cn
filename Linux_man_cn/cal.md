# cal

## 说明

**cal命令** 用于显示当前日历，或者指定日期的日历

## 选项

```markdown
cal [选项] [[[日] 月] 年]

-1, --one        只显示当前月份(默认)
-3, --three      显示上个月、当月和下个月
-s, --sunday     周日作为一周第一天
-m, --monday     周一用为一周第一天
-j, --julian     输出儒略日,即显示“julian”日期
-y, --year       输出整年

```

## 实例

```bash
cal -j
cal 9 1752  # 显示指定月，年的日历

```
