# **madadm**

常见Raid类型及基本特性,n代表磁盘数量
| Raid级别 | Raid0 | Raid1 | Raid5 | Raid6 | Raid10 |
| :--: | :--: | :--: | :--: | :--: | :--: |
| 别名 | 条带 | 镜像 | 分布式奇偶校检条带 | 双重奇偶校检条带 | 镜像加条带 |
| 磁盘数 | n≥1 | 2n(n≥1) | n≥3 | n≥4 | 2n(n≥2)≥4 |
| 可用容量 | 全部 | 50% | (n-1)/n | (n-2)/n | 50% |
| 最大容错 | 0 | n-1 | 1 |  | n/2 |
| 读性能 | n | n | n-1 |  | n |
| 写性能 | n | 1 | n-1 |  | n/2 |
